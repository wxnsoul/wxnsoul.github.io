2020-05-06

爬虫

#中国大学排名网站链接

  • http://www.zuihaodaxue.cn/zuihaodaxuepaiming2016.html

#需求描述

  • 输入:大学排名URL链接
  • 输出:大学排名信息的屏幕输出(排名，大学名称，总分)
  • 技术路线:requests-bs4
  • 定向爬虫:仅对输入URL进行爬取，不扩展爬取。
  • 目前只能实现爬取静态网页

#技术流程

  • 前提准备:首先查看robots页面,发现并没有内容,所以说明我们的爬取时合法的
  • 步骤1:从网络上获取大学排名网页内容 (getHTMLText())
  • 步骤2:提取网页内容中信息到合适的数据结构 (fillUnivList()存储到列表中)
  • 步骤3:利用数据结构展示并输出结果(printUnivList()打印输出)

  • `import bs4
    import requests

def getHTMLText(url):
try:
r = requests.get(url, headers=head, timeout=30)
r.raise_for_status() # 如果不是200就引起错误
r.encoding = r.apparent_encoding
print(“页面获取成功”)
return r.text
except:
print(“页面获取错误”)

def fill_univ_list(ulist, html): # 核心
soup = BeautifulSoup(html, ‘html.parser’)
for tr in soup.find(‘tbody’).children:
if isinstance(tr, bs4.element.Tag):
tds = tr(‘td’) # tds=tr.find_all(‘td’)
ulist.append([tds[0].string, tds[1].string, tds[2].string]) # 将提取到的信息存到列表中

def print_univ_list(ulist, num):
a = “{0:^10}{1:{3}^10}{2:^10}”#^10就是与上一个输出的分隔长度
print(a.format(“排名”, “学校”, “地域”, chr(12288)))#使用chr(12288)中文空格填充解决对不齐的问题
for i in range(num):
u=ulist[i]
print(a.format(u[0], u[1], u[2], chr(12288)))

def main():
uinfo = []
url = “http://www.zuihaodaxue.com/zuihaodaxuepaiming2016.html"
html = getHTMLText(url)
fill_univ_list(uinfo, html)
print_univ_list(uinfo, 20)

  • 遇到了很多问题,比如说定义函数需要与之前的内容空开两行,第二个参数与前一个参数之间需要多加一个空格,函数名称得小写等等

  • 还有一个查找得错误if isinstance(tr, bs4.element.Tag)这句话我给tr加了个引号,,,导致一直找不到内容
  • 最后的输出内容
  • 解释:)观察页面源代码可以发现所有的排名信息都在tbody标签中,所以取出tbody标签得所有子标签,并进行循环.每一个tr标签中都是一个大学,tr中的每一个td都是一个信息(排名,大学名称,地域,总分等等),所以用isinstance进行判断,如果是tr标签,那就将其中得td标签取出,并放在ulist中进行储存

#代码优化

  • 中文对齐问题得原因当中文字符宽度不够时,采用西文字符进行填充,但是中西文字符占用宽度是不同得,所以导致了不能对齐得问题
  • 问题解决char(12288)采用中文字符空格填充
  • print("{0:^10}{1:{4}^10}{2:^20}{3:^10}".format(u[0], u[1], u[2], u[3], chr(12288)))#^10就是与上一个输出的分隔长度,{4}就是指chr(12288)
