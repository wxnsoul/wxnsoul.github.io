#信息的标记

  • 如果给你某样物品(a)的一个信息,你会直接知道在说a的什么信息,但是如果给了你一堆关于a的信息,那么你就会需要想一想这个信息是a的名字?a的年份?所以我们就需要信息标记来更好的帮我们处理信息

#国际公认的三种信息标记种类

  • XML,JSON,YAML
  • XML叫扩展标记语言,是一种和html很接近的标记语言))历史上先有HTML后又XML,所以可以说XML是基于HTML发展以来的,甚至可以说是HTML语言的拓展
    如果有内容,就使用<name></name>,没有就使用</html>,也可以用<!-- -->来添加注释
  • JSON JavsScript Object Notation(Java语言中对面向对象信息的一种表达方式)
  • 简单讲就是有类型的键值对构建的信息表达方式 JSON的好处就是可以直接把JSON格式作为程序的一部分,大大简化程序
  • YAML
  • 和JSON最大的区别就是它采用无类型的键值对形式保存数据)
    可以看到没有双引号了,所以我们常用缩进的方式来显示所属关系)用减号’-‘来表达并列关系)用竖线’|’来表达一块内容,’#’来表达注释内容

#三种信息标记形式的比较

  • XML用尖括号,标签来表达信息
  • JSON用有类型的键值对
  • YAML用无类型的键值对
  • XML实例可以看到大部分内容被标签占据
  • JSON示例
  • YAML实例
  • XML最早的通用信息标记语言,可拓展性好,但是繁琐(可以说信息标记的通用格式都是发展于XML)
  • JSON信息有类型，适合程序处理(js)，较XML简洁。
  • YAML信息无类型，文本信息比例最高，可读性好。
